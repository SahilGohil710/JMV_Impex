# **App Name**: CopperCraft Showcase

## Core Features:

- Item Catalog: Interactive catalog displaying copper items.
- Detailed Product Views: Dedicated pages providing comprehensive information on each product.
- Advanced Filtering: Filter and sort products based on properties like material, size, and purpose.
- Simplified Inquiry Process: An integrated contact form and contact details, so customers can initiate an order with ease.

## Style Guidelines:

- Primary color: Copper-inspired HSL(24, 67%, 54%) converted to RGB Hex: #D48853 to capture warmth and earthiness.
- Background color: Desaturated earthy tone HSL(24, 20%, 95%) converted to RGB Hex: #F5F0ED for a natural canvas.
- Accent color: Patina-esque green HSL(54, 41%, 54%) converted to RGB Hex: #A2A853 to evoke aged copper.
- Body and headline font: 'Alegreya' serif, with its elegant, intellectual, contemporary feel. Note: currently only Google Fonts are supported.
- Use of simplistic line icons colored with the patina-esque accent to represent product categories and website functions.
- Grid-based layout with ample spacing for showcasing products clearly, with key product specs below.
- Subtle transitions when loading product pages.
